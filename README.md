# Accident Risk Predictor 🚦

An AI-powered web application that predicts traffic accident risk levels using Machine Learning.

## Features

* Predicts accident risk levels (Low / Medium / High)
* Interactive React frontend
* Flask backend API
* Machine Learning integration using Scikit-learn
* Real-time prediction system
* Traffic and weather parameter analysis

## Tech Stack

### Frontend

* React.js
* CSS

### Backend

* Flask


### Machine Learning

* Scikit-learn
* Pandas
* NumPy

## Project Structure

```text
backend/
frontend/
mlprojfinal.ipynb
```

## How to Run

### Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Runs on:

```text
http://127.0.0.1:5000
```

### Frontend

```bash
cd frontend
npm install
npm start
```

Runs on:

```text
http://localhost:3000
```

## Machine Learning Model

The project uses:

* Random Forest Classifier
* KMeans Clustering
* StandardScaler
